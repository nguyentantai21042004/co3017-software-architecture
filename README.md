# CO3017 - Software Architecture

## Giới thiệu

Đây là repository cá nhân chứa toàn bộ tài liệu, code và notes cho môn học **CO3017 - Software Architecture**.

## Cấu trúc repository

### 1. Overleaf (`overleaf/`)
Chứa tất cả file LaTeX để sync với Overleaf:
- `current-project/` - Project hiện tại (main.tex, sections)
  - `main.tex` - File chính LaTeX
  - `sections/` - Các phần nội dung
    - `course-overview.tex` - Tổng quan môn học
    - `curriculum-content.tex` - Nội dung chương trình
    - `schedule-details.tex` - Lịch trình chi tiết
    - `study-guide.tex` - Hướng dẫn học tập
- `templates/` - Templates LaTeX cho các loại tài liệu
  - `assignments/` - Templates cho bài tập
  - `reports/` - Templates cho báo cáo  
  - `presentations/` - Templates cho slide thuyết trình
  - `thesis/` - Templates cho luận văn/đồ án
- `common/` - Styles, packages và macros dùng chung

### 2. Assignments (`assignments/`)
Chứa các bài tập và dự án môn học:
- `assignment-1/` - Bài tập thực hành 1 (Tuần 7)
- `assignment-2/` - Bài tập thực hành 2 (Tuần 8)
- `final-assignment/` - Assignment cuối kỳ (Deadline: 07/12/2025)
- `presentations/` - Thuyết trình dự án (Tuần 15)

### 3. Study Materials (`study-materials/`)
Chứa tài liệu học tập và nghiên cứu:
- `solid-principles/` - Các nguyên tắc SOLID
- `architectural-patterns/` - Các mô hình kiến trúc
- `design-patterns/` - Design patterns
- `case-studies/` - Các case study thực tế
- `advanced-topics/` - Chủ đề nâng cao (IoT, Web3, MLOps, Digital Twin)

### 4. References (`references/`)
Chứa tài liệu tham khảo:
- `books/` - Sách giáo khoa về Software Architecture
- `papers/` - Bài báo khoa học về kiến trúc phần mềm
- `slides/` - Slide bài giảng
- `documentation/` - Tài liệu về ADRs và documentation patterns

## Cách sử dụng

1. **Overleaf**: Copy toàn bộ folder `overleaf/` vào Overleaf project để làm việc với LaTeX
2. **Assignments**: Lưu các bài tập và dự án vào thư mục `assignments/` tương ứng
3. **Study Materials**: Tổ chức tài liệu học tập theo các chủ đề trong `study-materials/`
4. **References**: Lưu sách, papers, slides vào `references/` theo danh mục

## Quy ước

- Mỗi assignment nên có README.md mô tả yêu cầu và cách thực hiện
- LaTeX templates nên có documentation rõ ràng về cách sử dụng  
- Study materials nên được tổ chức theo tuần học hoặc chủ đề
- ADR (Architecture Decision Records) nên được lưu trong `references/documentation/`
