# Tasks: Video Presentation Recording

This checklist is a guide for the user to record the presentation using the provided script.

- [ ] Preparation
    - [ ] Open `presentation/slides.md` (or the rendered PDF).
    - [ ] Open `presentation/video_script_vi.md` (The Script).
    - [ ] Setup screen recording software (OBS, Screenflick, etc.).
- [ ] Recording Session
    - [ ] **Section 0: Intro**: Read script for Slide 1.
    - [ ] **Section 1: General**: Read script for Slides 2-5 (Vision, Challenges, Solution).
    - [ ] **Section 2: Context**: Read script for Slides 6-8 (Business/Technical Context).
    - [ ] **Section 3: Stakeholders**: Read script for Slide 9-10.
    - [ ] **Section 4: Requirements**: Read script for Slide 11-12 (Functional) & 13-14 (Non-Functional).
    - [ ] **Section 5: Design**: Read script for Slide 15-16 (Style) & 17-19 (ADRs).
    - [ ] **Section 6: Views**: Read script for Slide 20-22 (Module, Component, Allocation).
    - [ ] **Section 7: Behavior**: Read script for Slide 23-24 (Sequence Diagrams).
    - [ ] **Section 8: SOLID**: Read script for Slide 25.
    - [ ] **Section 9: Implementation**: Read script for Slide 26-27 (MVP Status).
    - [ ] **Section 10: Evaluation**: Read script for Slide 28.
    - [ ] **Section 11: Demo**: narrate the live demo as per the script guide (Slides 30-38).
    - [ ] **Section 12: Conclusion**: Read script for Slide 39-40.
